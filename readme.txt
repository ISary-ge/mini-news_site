--Внутренние
    Статьи добавлять в internal_articles.php. После последней фигурной скобки '}' запятую ставить не надо
    В поле content не использовать двойные кавычки. Можно заменить на ' или на «»
    Вместо переноса строки юзать <br>

--Внешние
    Статьи добавлять в external_articles.php. После последней фигурной скобки '}' запятую ставить не надо
    Чтобы вставлять значение get-переменной, необходимо использовать макрос {имя get-переменной}. Например: значние get-переменной clickid=qwerty123 в целевом url будет добавлятся вместо макроса {clickid}

--Видео статьи
    Статьи добавлять в video_articles.txt

Частоту показа тизеров можно менять в файле get_articles.php(переменная $periodicity)
Рандомный показ тизеров можно включать/отключать в файле get_articles.php(переменная $random)
